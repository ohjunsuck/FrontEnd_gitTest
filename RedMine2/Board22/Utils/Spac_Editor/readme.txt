[HTML 에디터 안내]
* 케릭터셋은 UTF-8 입니다.
* 에디터는 자바스크립트와 HTML로 만들어졌습니다.
* Spac Editor는 각종 데스크탑용 브라우저의 자체 내장된 에디터기능을 사용합니다.

[업로드/이미지브라우저 안내]
* 이미지 업로드/이미지브라우저 기능은 PHP로 만들어졌습니다.
* 파일업로드기능을 사용하기 위해서는 Spac_Editor/upload 디렉토리가 쓰기 가능해야 합니다.(예.707)
* spac_editor.class.php 파일의 initConfig 메소드내 초기화 변수의 수정을 통해 다양한 설정이 가능합니다. (접속제한, 회원별자료실제공, 썸네일사이즈조절 등)

[설치안내]
* Spac Editor의 압축파일을 웹사이트(www.phpwork.kr) 에서 내려받아 웹디렉토리의 루트에 압축을 해제합니다.
* 브라우저로 샘플페이지를 접속하여 테스트 합니다. ( www.yourdomain.com/Spac_Editor 접속)
* 에디터를 삽입할 페이지에 아래와 같이 에디터삽입 코드를 넣습니다. ("/Spac_Editor/index.html" 참조)

------------------------------------------------------------------------------------------
<script language="Javascript" src="/Spac_Editor/spac_editor_common.js"></script>
<script language="Javascript" src="/Spac_Editor/spac_editor.js"></script>
<link rel="stylesheet" type="text/css" href="/Spac_Editor/spac_editor_common.css">
<link rel="stylesheet" type="text/css" href="/Spac_Editor/spac_editor.css">
<script language="Javascript">createEditor('contents','높이픽셀값','초기내용','제외할메뉴아이콘을 콤마를 이용해 나열');</script>
------------------------------------------------------------------------------------------

ex1) 모든아이콘을 보이게 할때
    <script language="Javascript">createEditor('contents','높이픽셀값','초기내용','');</script>

ex2) 이미지업로드아이콘, 테이블 아이콘을 안보이게 할때
    <script language="Javascript">createEditor('contents','높이픽셀값','초기내용','image,table');</script>

ex3) 소스보기 아이콘만 보이게하고, 다른아이콘들은 모두 안보이게 할때
    <script language="Javascript">createEditor('contents','높이픽셀값','초기내용','all');</script>

ex4) 소스보기 아이콘만 안보이게하고, 다른아이콘들은 보이게 할때
    <script language="Javascript">createEditor('contents','높이픽셀값','초기내용','all');</script>

ex5) 모든 아이콘 출력안하고 소스보기체크박스도 출력 안하고자할때 (모든 아이콘 감추기)
    <script language="Javascript">createEditor('contents','높이픽셀값','초기내용','source,all');</script>
